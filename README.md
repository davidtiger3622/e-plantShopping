# e-plantShopping

Paradise Nursery Shopping Application - A React-based e-commerce platform for browsing and purchasing houseplants.

## Project Description

Paradise Nursery is a front-end shopping application that allows users to browse a variety of houseplants organized by categories and add them to a shopping cart. The application features a landing page, product listing page, and shopping cart with full cart management functionality.

## Features

- Landing page with company information and background image
- Product listing with multiple plant categories
- Shopping cart with quantity adjustment
- Dynamic cart icon showing total items
- Redux state management for cart functionality
- Responsive design

## Technologies Used

- React
- Redux Toolkit
- React Router
- Vite
- CSS3

## Installation
```bash
git clone https://github.com/davidtiger3622/e-plantShopping.git
cd e-plantShopping
npm install
npm run dev
```
